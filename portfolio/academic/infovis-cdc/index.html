About the Project

This project focused on visualization on the CDC public data of youth obesity from across 50 states.

We feel youth obesity is an important and interesting issue to explore because it impacts many kids and parents in the country. By creating useful visualization, we wanted to help parents, school administrators and others to learn more about the school health programs.

Phase 1: Research

Defining the Problem Space

CDC Division of Adolescent and School Health have made public data and statistics of child obesity. The data is currently available for all 50 US states spanning across 20 dimensions and 3 years (2007, 2005, and 2003). But since the data is all in static PDF files (Example PDF), it is difficult for health science researchers or other interested third parties to browse and explore the data.

Understanding Users

Our visualization system is built to support the following target audience: School board administrators, Professionals in State and Local Health Agencies, Health/nutrition researchers, Health Scientist, Health Education Professionals and Physical Education Professionals.

Phase 2: Visualizations

Fish Eye View

  • The data in Excel is transformed into a color coded representation augmented with FishEye display. Each data is assigned a RGB color component.
  • Data that varies from 0-25 have a high 'Blue' component. Data that varies from 30-60 have a high 'Green' component and data that varies from 70-100 have a high 'Red' component.
  • The user now has the ability to find outliers and search for patterns in the dataset. Further, we have the bar fill option, that displays the color coded information with the magnitude of the data as a bar column. This is useful in seeing at a glance the variation in magnitude of the data and the actual magnitude value.

Map View

  • Another view of our system is the Map View, an improvement and can be thought of as an extension on top of parallel coordinates. The 20 dimensions are grouped via the student and school variables.
  • By leveraging the convenience of parallel coordinates and improving it via separation of data from the dimension, we believe we have a fun, easy, dynamic, explorative and a browsable interactive map for the CDC child obesity data.

Scatter Plot Matrix View

  • This provides an overview of correlation between all data in a given year.
  • The ability to explore distribution of large data-set via filters.
  • The user can hover on any single point on any cell in the matrix. This point in RED gets reflected and highlighted on every cell in the matrix.
  • The search box allows the user to enter a state name. This applies a dynamic filter in real time across the data and displays those states that satisfy the search query.
