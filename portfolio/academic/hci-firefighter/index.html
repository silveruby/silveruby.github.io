HCI Research & Design - AR Heads-Up Display

User Research, Usability Testing, Interaction Design

About the Project

This project focused on solving a social problem with technology using User-Centered Design process. There were four main parts to this project: defining a problem (PDF), creating design alternatives, doing usability study (PDF), and conducting evaluation (PDF).

Our goal was to design a system that would allow firefighters to navigate quickly through low visibility scenarios.
